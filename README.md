# Microserviço de autenticação com Nodejs

## Composição do projeto
Neste projetos temos alguns **Endpoints Base** que podem ser extendidos de forma mais adequada para outro contexto.

São eles: 

### Usuários

* GET /users
* GET /users/:uuid
* POST /users
* PUT /users/:uuid
* DELETE /users/:uuid

### Autenticação

* POST /token
* POST /token/validate
